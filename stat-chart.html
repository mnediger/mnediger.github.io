Designing a responsive, flexible infographic chart widget

Venngage | 2021

The context

Venngage is a graphic design tool that specializes in infographics

Venngage is a web-based SaaS graphic design tool. Many of Venngage’s core users use the tool to create documents like infographics, reports, presentations, and diagrams to aid communication at work.

Many of these designs feature data, visualized in charts and tables. Users often cite Venngage’s data visualization capabilities as a key reason they choose Venngage over similar competitors. With that in mind, we want to ensure that our data-focused templates are as easy to edit as possible.

The opportunity

Identifying an opportunity to streamline the editing flow for infographic designs

Through user interviews and qualitative research into our templates and user designs, I identified a common use case where the editing flow was tedious but the users’ desired output was consistent and predictable: visualizing a single statistic, a percentage with a simple chart, some text, and often an icon.

This was a good opportunity to simplify the user experience without compromising on the quality of the resulting designs.

The pain points

This type of simple visualization is a feature of many user designs and templates. For users to edit designs with these visuals, they had to:

  • Input their data into a text widget and a chart widget separately
  • Adjust the size and position of each widget to work with their content
  • Readjust to work with any styling changes to the content or chart type

As a result, most users stuck very closely to the template design, even if that wasn’t ideal for their content, or how they wanted to display their data.

The requirements

The widget must be flexible enough in style and structure to replicate the way this type of data is represented in existing templates, all while responding to the user’s content. This includes variations in:

  • Chart type (donut, half-donut, progress bar, pictogram)
  • Layout
  • Chart and text style
  • Icon

The design

  • The widget has 4 components: value label, icon label, chart, and description text label. Each option, except the chart, can be toggled on or off.
  • The widget responds to the users content and style choices - resizing and repositioning the labels as necessary to fit within the chart.
  • The editing is WYISWYG, like other Venngage widgets. Users select the widget or components within the widget to edit via the toolbar.
  • The user can choose from a few layout options and easily swap out the chart type (donut, half-donut, progress bar, icon chart). Styles are maintained so the editing required from the user is minimal.

The results

Since its release, this widget has become Venngage’s most-used chart widget.

Users who use the new widget complete their designs 24% more than those who used the old freeform components.
